# 数字生命卡片

一张可自定义的 Agent 运行时框架。**换配置文件即换身份。**

---

## 这是什么

一个完整的、可本地运行的 Agent 技能包模板。它包含：

- **纯计算引擎** — 实体管理、修饰符系统、阈值检测、叙事渲染（`scripts/engine/`）
- **记忆管道** — 对话提取、情景记忆、知识图谱（`scripts/soli_memory/`）
- **功能模块** — 保险库（RSA-4096）、读书追踪、日记系统
- **运行时状态** — 原子写入、BOM 兼容、备份轮转
- **121 项单元测试** — 全过，禁用词零命中

**所有角色设定和叙事内容已清空为白板**——填入你自己的身份、参数和叙事文本即可启动一张全新的数字生命卡片。

---

## 快速开始

1. 克隆本仓库到你的 Agent 技能目录
2. 编辑 `SKILL.md` 末尾的五个模板区域（身份/参数/准则/系统/道具）
3. 填充 `scripts/cn_narratives/` 和 `data/engine/narratives.json` 中的白板叙事
4. 部署到兼容的 Agent 平台（Trae / WorkBuddy / OpenClaw）

```bash
# 加载卡片
python scripts/dataLoading.py

# 运行引擎测试
python -m unittest discover -s tests/engine
```

## 目录结构

```
scripts/
├── engine/           ← 纯计算引擎（7 .py, 703 行, 0 域词汇）
├── soli_memory/      ← 记忆管道（chatlog + distil）
├── cn_narratives/    ← 叙事文本库（白板，待填充）
├── soul_core.py      ← 核心状态机
├── soul_sense.py     ← CLI 入口
├── dataLoading.py    ← 数据加载 + 身份注入
└── tools/            ← 开发工具

tests/engine/         ← 121 项单元测试（全过）
data/engine/          ← 4 组引擎配置 JSON
MEMORY/               ← 记忆存储（空，已清空）
```
